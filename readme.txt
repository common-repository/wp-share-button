	Awesome Share Button

== Description ==


Displaying social share button on your website is most powerful way to boost your website traffic and visitors from social media sites, best way to recommend website content to social media friends.

<strong>WP Share Button</strong> is one of best plugin to display social share buttons on your website easy to manage and customizing via settings page.

<strong>Plugin Features</strong>

* 35 Different ready themes.
* Use via short-codes.
* Ajax share count.


### WP Share Button by http://pickplugins.com

<br>
<strong>Premium Add-ons</strong>

* [Total Count &raquo;](http://www.pickplugins.com/item/wp-share-button-total-count/)

<strong>How to display share buttons</strong>

In your theme files inside post loop just add following php code

`<?php echo do_shortcode('[wp_share_button]'); ?>`

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>WP Share Button</strong>" activate it.<br />
3. Go seetings(WP Share Button Settings) page and changes your seetting and save.
